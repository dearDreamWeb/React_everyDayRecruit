# 天天直聘
基于React Hooks、Express、SocketIo开发的移动端的webApp，名为天天直聘。  
除了基础的登录、注册等功能外，还实现了基于SocketIo的实时聊天功能，聊天功能还添加了表情功能。   
React开启的端口：localhost:8080  
Express开启的端口：localhost:3000  
SocketIo开启的端口：localhost:4000   

# 核心技术栈
- React Hooks
- Antd-mobile
- Express
- Mysql
- SocketIo
